Smart Road Pothole Detection System

Sadak Saathi is an AI-powered solution designed to detect road potholes using image and sensor data, helping authorities repair damaged roads faster and prevent accidents.

How It Works

  • Mobile app captures road images automatically
  • AI model detects potholes with high accuracy
  • GPS tagging marks exact location
  • Data sent to centralized dashboard
  • Authorities receive real-time alerts

Key Features

  • Automatic detection using Computer Vision
  • Heatmap of high-risk road zones
  • Priority-based repair tracking
  • User reporting & verification
  • Admin monitoring dashboard

Expected Impact

Reduced road accidents, faster maintenance response, improved road quality, and a smarter city-level infrastructure monitoring system.
